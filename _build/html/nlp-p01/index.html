sou = pd.read_csv('data/SOTU.csv')

Number of Speeches per President¶

# Hint - use value_counts() on the President column
# Hint - sort in order of dataframe
counts = sou["President"].value_counts(sort=False)
# Plot 
# Hint - use the .plot() method for Pandas Series, make sure all presidents show up on x-axis

counts.plot(kind="bar", figsize=(20, 10))
plt.title("Number of Speeches per President", fontsize=24)
plt.xlabel("President", fontsize=18)
plt.ylabel("Count", fontsize=18)
plt.xticks(rotation=90, fontsize=13)
plt.yticks(fontsize=13)

plt.show()
plt.savefig("outputs/num_speeches.png")
<Figure size 640x480 with 0 Axes>

It seems pretty interesting that each president seems to give more speeches each as we get closer and closer to the present. Maybe this might be a function of the party system and increasing tension between parties so more speeches are necessary for communication.

Number of Speeches per Year¶

# Hint - Use value counts and sort by years
counts_by_year = sou["Year"].value_counts().sort_index()

plt.figure(figsize=(5.75, 5))
counts_by_year.plot(kind="line")

plt.title("Number of State of the Union Speeches per Year")
plt.xlabel("Year")
plt.ylabel("Count")

plt.tight_layout()
plt.show()
plt.savefig("outputs/num_speeches_year.png")
<Figure size 640x480 with 0 Axes>

It seems like there were many more SOU speeches done in the beginning when the US was first founded. This makes sense as I would assume there were lots of speeches and presentations to create the laws to run the nation.

Word Count Distribution¶

# Hint - try seaborn.histplot()

plt.figure(figsize=(5.75, 5))

sns.histplot(
    sou["Word Count"],
    bins=18,
    kde=False,
    color="steelblue"
)

plt.title("Distribution of State of the Union Speech\nWord Counts")
plt.xlabel("Word Count")
plt.ylabel("Count")

plt.tight_layout()
plt.show()
plt.savefig("outputs/dist_word_counts.png")
<Figure size 640x480 with 0 Axes>

It’s quite interesting how many of the speeches are around 5000 words in length which is quite short for a speech. I would have imagined most of the speeches are closer to 15000. However, this makes sense. I would assume the speeches would be longer in situations where the nation is under threat or actively involved in something which is a threat to national security.

Word Count Distribution over Year¶

# Hint: try seaborn.rugplot()
plt.figure(figsize=(6, 5))

sns.scatterplot(
    data=sou,
    x="Word Count",
    y="Year",
    s=40
)

sns.rugplot(
    data=sou,
    x="Word Count",
    height=0.02,
    alpha=0.6,
)

sns.rugplot(
    data=sou,
    y="Year",
    height=0.02,
    alpha=1,
    color=(59/255, 117/255, 174/255)
)

plt.title("Speech Year Versus Word Count")
plt.xlabel("Word Count")
plt.ylabel("Year")

plt.tight_layout()
plt.show()
plt.savefig("outputs/speech_vs_word_count.png")
<Figure size 640x480 with 0 Axes>

It seems that as we move towards modern times, the word count really starts to shrink. This also makes it appear as 1820-1900s had the US in many disagreements and things which might explain why the majority of the speeches during that time period have word counts greater than 5000.

Word Count Distribution per President¶

# Hint: use pandas groupby to get mean word count per president then sort by order

avg_words = sou.groupby("President")["Word Count"].mean()

avg_words = avg_words.loc[sou["President"].unique()]
plt.figure(figsize=(10, 6))
avg_words.plot(kind="bar")

plt.title("Average State of the Union Word Count\nper President")
plt.xlabel("President")
plt.ylabel("Average Word Count")

plt.xticks(rotation=90)
plt.tight_layout()
plt.show()
plt.savefig("outputs/avg_sou_wc_president.png")
<Figure size 640x480 with 0 Axes>

It seems like there are like two main discontinuities where from Biden to Wilson, the majority of the speeches are short but Taft to Washington has significantly more words. This could be an explanation as to how the nation’s priorities have shifted and hence some topics needed more explanations than others.
